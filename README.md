# Disease Discussion Blog

A modern, fast, and responsive web application built with **Next.js**, **Redux Toolkit**, **Tailwind CSS**, and **TypeScript** that allows users to create, edit, and delete posts discussing various diseases. The app is designed to foster community engagement by enabling users to ask questions, share information, and support each other.

## Features

- **User Authentication** (optional, can be added)
- Create new posts about diseases
- Edit and delete existing posts
- Read detailed posts with expandable content
- Responsive UI designed with Tailwind CSS
- State management using Redux Toolkit for predictable state updates
- Commenting system on posts (optional/extendable)

## Tech Stack

- **Next.js** - React framework for server-side rendering and static site generation  
- **TypeScript** - Strongly typed JavaScript for better developer experience and fewer bugs  
- **Redux Toolkit** - Simplified and powerful state management solution  
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development  
- **React** - UI library used by Next.js
